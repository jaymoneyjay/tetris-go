# tetris-go
A go implementation of the classic tetris game

![Screenshot](Screenshot.png)

## Installation
```bash
$ git clone https://github.com/jaymoneyjay/TetrisGo.git
$ go get github.com/nsf/termbox-go
```

## Run
```bash
$ cd TetrisGo
$ go build .
$ ./TetrisGo
```
## Instructions
```bash
q: quit
right: move right
left: move left
down: move down
up: rotate
```
